## 3. Definition of Ready (DoR)
Una historia se considera lista cuando:
- Está descrita con claridad.
- Tiene criterios de aceptación.
- Tiene diseño asociado (prototipo).
- No depende de otra historia.

---

## 4. Definition of Done (DoD)
Una historia está completada cuando:
- El código está implementado.
- Tiene pruebas básicas exitosas.
- La funcionalidad se valida en el prototipo.
- Se incluye en el documento maestro como evidencia.
- Está integrada al repositorio.
